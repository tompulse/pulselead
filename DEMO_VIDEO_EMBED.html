🎥 Démo : Intégrations Vidéo

1. Vidéo hébergée localement (votre fichier MP4)

✅ La vidéo est sur votre serveur
❌ Problème : Fichier trop gros pour GitHub (44 MB)

2. Vidéo YouTube embed (exemple)

✅ La vidéo s'affiche DIRECTEMENT sur votre page
✅ Le visiteur ne voit jamais YouTube.com
✅ Autoplay, mute, loop - tout fonctionne
✅ Chargement ultra-rapide, bande passante illimitée

3. Vidéo Cloudinary (recommandé)

✅ Vidéo hébergée gratuitement sur CDN rapide
✅ S'affiche comme si elle était sur votre serveur
✅ Pas de logo YouTube
✅ Contrôle total sur le player

Ouvrez ce fichier dans votre navigateur pour voir les 3 options
