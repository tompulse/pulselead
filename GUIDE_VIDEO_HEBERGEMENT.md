# 🎥 Guide : Héberger votre vidéo pour la landing page

## Problème
Le fichier vidéo (44 MB) est trop gros pour GitHub. Il faut l'héberger ailleurs.

## ✅ Solution recommandée : Cloudinary (Gratuit)

### Étape 1 : Créer un compte Cloudinary

1. Allez sur : https://cloudinary.com/users/register_free
2. Inscrivez-vous (gratuit, pas de CB requise)
3. Confirmez votre email

### Étape 2 : Upload votre vidéo

1. Une fois connecté, cliquez sur **"Media Library"**
2. Cliquez sur **"Upload"** (bouton bleu)
3. Sélectionnez votre fichier `demo-pulse.mp4`
4. Attendez la fin de l'upload

### Étape 3 : Récupérer l'URL

1. Cliquez sur la vidéo uploadée
2. À droite, vous verrez **"URL"**
3. **Copiez l'URL complète** (elle ressemble à : `https://res.cloudinary.com/votre-cloud/video/upload/v123456/demo-pulse.mp4`)

### Étape 4 : Donnez-moi l'URL

Une fois que vous avez l'URL, **donnez-la moi** et je modifie le code pour l'intégrer !

---

## Alternative : Bunny CDN (Plus technique mais ultra-rapide)

Si vous voulez un CDN professionnel :
1. Créez un compte sur https://bunny.net
2. Créez une "Storage Zone"
3. Uploadez votre vidéo
4. Récupérez l'URL publique

---

## Autre alternative : YouTube Embed (Plus simple)

**Important** : Avec YouTube embed, la vidéo s'affiche DIRECTEMENT sur votre landing page.
Le visiteur ne voit jamais YouTube, c'est intégré comme si c'était votre vidéo.

1. Uploadez sur YouTube en "Non répertoriée"
2. Récupérez le lien
3. Je l'intègre dans la page

**Le visiteur verra juste la vidéo jouer automatiquement sur votre page, rien d'autre !**
