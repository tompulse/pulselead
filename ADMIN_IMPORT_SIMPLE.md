# 🚀 Import CSV Simple - En 2 Clics

## ✅ Configuration (1 fois)

1. **Supabase SQL Editor** → Exécuter `SETUP_ADMIN_TOMIO.sql`
2. C'est tout !

## 📥 Import hebdomadaire (2 clics)

1. **Aller sur** `/admin/import`  
2. **Upload CSV**  
3. **Cliquer "Importer"**  
4. ✅ Terminé !

## Format CSV

```
siret;Entreprise;date_creation;siege;categorie_juridique;categorie_entreprise;complement_adresse;numero_voie;type_voie;libelle_voie;code_postal;ville;coordonnee_lambert_x;coordonnee_lambert_y;code_naf
```

Séparateur : **point-virgule** (;)
